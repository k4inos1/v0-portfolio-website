# GitHub Copilot Automation Configuration

## 🤖 Copilot Automation Rules for portfolio-k4inos

### PR Review Automation

**Triggers:**
- `review: auto` → Genera automáticamente sugerencias
- `performance: check` → Verifica performance
- `seo: check` → Verifica SEO
- `accessibility: check` → Verifica a11y

### Auto-Merge Rules

```yaml
Auto-merge cuando:
  - ESLint pass ✅
  - Build success ✅
  - Tests pass ✅
  - Performance OK ✅
```

### Focus Areas

```yaml
Portfolio-specific checks:
  ✅ Performance (Lighthouse)
  ✅ SEO optimization
  ✅ Accessibility (WCAG)
  ✅ Mobile responsiveness
  ✅ Image optimization
  ✅ Code splitting
```

### Copilot Commands

```
@copilot check: performance        # Lighthouse check
@copilot check: seo                # SEO check
@copilot check: accessibility      # a11y check
@copilot optimize: images          # Image optimization
```
